# API Java – Gestão de Usuários e Interesses Profissionais

API RESTful desenvolvida em Java com Quarkus, focada no gerenciamento de usuários, seus interesses profissionais e regras de negócio específicas.
Este backend serve como suporte para o site onde o usuário pode:

- Se cadastrar
- Realizar testes
- Visualizar áreas de atuação
- Registrar interesses profissionais
- Obter clareza sobre objetivos pessoais e profissionais

A API garante consistência dos dados, segurança, regras de negócio e comunicação eficiente com o banco Oracle.

## Visão Geral

Nossa API atua como a camada de backend para um sistema focado em orientação profissional.
Ela permite armazenar, consultar, atualizar e remover informações sobre usuários e seus interesses de carreira, oferecendo:

- CRUD completo
- Regras de negócio específicas
- Login com email e senha
- Integração pronta para qualquer frontend (web, mobile ou desktop)
